## 一、构造方法

### 1、字面量直接构造

```python
d1 = {}                    # 空 dict
d2 = {'a': 1, 'b': 2}      # 键值对字面量
d3 = {                     # 多行可读性高
    'name': 'Alice',
    'age': 30
}
```

* 键必须是 **可哈希** 对象；值任意。
* 重复键以 **最后一次** 为准。
