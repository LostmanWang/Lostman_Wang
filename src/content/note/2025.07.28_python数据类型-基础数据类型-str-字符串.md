## 一、字符串的表示方式

### 1、单引号: '允许包含有" 双" 引号'

### 2、双引号: " 允许嵌入 '单' 引号"

### 3、三重引号: '''三重单引号''', """ 三重双引号"""

* 使用三重引号的字符串可以跨越多行——其中所有的空白字符都将包含在该字符串字面值中。

  ```python
  v1 = "包治百病"
  v2 = '包治百病'
  v3 = "'包'治百病"
  v4 = '"包"治百病'
  v5 = """
  吵架都是我的错，
  因为大家打不过。
  """
  ```
